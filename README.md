# AI-Affiliate-Auto

🚀 **AI를 활용한 저예산 어필리에이트 자동화 플랫폼**  
Gemini Pro, Claude Pro, Grok Pro를 연동해 고품질 콘텐츠·코드·툴을 자동 생성하고, Amazon Associates 어필리에이트로 수익화하는 오픈소스 프로젝트입니다. Railway Hobby로 웹 앱 배포까지.

## 목적 & 비전
- 월 $80 AI 구독 비용만으로 자체 수익 모델 구축  
- 콘텐츠 생성 → 어필리에이트 링크 삽입 → 트래픽 유치 → 수익 발생  
- 목표: 3개월 내 월 $500+ 수익, GitHub 스타 500+  

## 주요 기능 (계획 중)
- AI 기반 블로그 포스트 자동 생성 (Claude/Gemini/Grok 통합)  
- Amazon 제품 추천 링크 자동 삽입  
- 간단 웹 앱/툴 호스팅 (Railway 배포)  
- GitHub Actions로 자동화 파이프라인 (향후)
